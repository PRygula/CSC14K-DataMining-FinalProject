CSC 145K - 291A Mining the Social Web Sentiment Analysis Project

Opinion Mining from the Twitter API

Project Overview

The research project will focus on Sentiment Analysis/Opinion Mining. This is a group project of teams size 3-4 (no more). Each team will introduce sentiment analysis and discuss the current approaches for computing sentiment in the context of social media. The research will also outline the main problems with sentiment analysis. From the list of problem, the research will propose a solution, provide a design and demo the implementation of the solution. The project is organized into three phases with points for each phase:

  • Planning and Design [ 40 points ]
  • Execution and Tracking [ 30 points ]
  • Presentation and Demo [ 30 points ]

Project Guidelines

Divide and conquer: Break the project down into multiple parts and follow the software architecture principle of separation of concerns. When designing your components encapsulate your functionality in each domain area.

Work Break Down Guidelines: These are only suggestions; it is your project so feel free to design your own implementation (as a team!). Data Mining includes multiple steps to acquire and deliver insightful information, for this project you may consider a separate component for each of the following:
  • Data Collection
  • Data Cleansing
  • Information Extraction
  • Creating Analytics
  • Visualization
